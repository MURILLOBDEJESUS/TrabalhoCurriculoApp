Instruções rápidas para habilitar o visualizador do currículo (PDF)

O site espera encontrar o arquivo do seu currículo com o nome exato `MURILO_Curriculo.pdf` dentro da mesma pasta onde está o `indexAPP.html`.

Passos:
1. Localize o seu PDF (ex.: `MURILO_BANDIERA DE JESUS _Curriculo.pdf`).
2. Renomeie o arquivo para `MURILO_Curriculo.pdf` (remova espaços e caracteres especiais).
3. Copie ou mova o arquivo para:
   `c:\Users\Murilo\Pictures\Murilo\currículo\`
4. Reabra o `indexAPP.html` no navegador. A seção "Currículo" deverá carregar o PDF embutido e os botões de download abrirão corretamente.

Se preferir que eu mova/renomeie o arquivo automaticamente, envie o PDF para a pasta do projeto ou autorize-me a realizar a mudança (na prática eu preciso que o arquivo exista dentro do workspace).
